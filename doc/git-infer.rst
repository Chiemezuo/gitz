``git infer``: Commit changes with an auto-generated message
------------------------------------------------------------

(from https://github.com/moondewio/git-infer)
USAGE
=====

.. code-block:: bash

    git infer [-a] [-A] [-h]
    
       -a    Include all tracked files
       -A    Include all files, even the untracked
       -h    See this help message
