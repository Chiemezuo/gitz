``git when``: When did each file change (date, commit, message), or
-------------------------------------------------------------------


Dotfiles are ignored by default
USAGE
=====

.. code-block:: bash

    git when [-a] [item [..item]]

Optional arguments
  ``-a``: Also include dotfiles
